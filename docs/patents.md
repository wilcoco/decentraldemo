# 참조 특허 요약

이 시스템의 안목 지수 계층(`src/weave/insight.js`)의 근거가 되는 두 특허.
원문 PDF는 `wilcoco/CAMS-KnowledgeNet` 저장소에 보관되어 있다.

## 등록특허 10-0913256 — 평가 방법

- 명칭: 다중 링크를 형성하는 정보 네트워크에서 링크 관계에 따른 대상 평가 방법
- 출원: 10-2005-0031125 (2005-04-14) / 등록: 2009-08-13 / 공고: 2009-08-24
- 발명자: 홍정수 / 당시 특허권자: 에스케이커뮤니케이션즈(주)

핵심 청구:
1. **(청구항 1)** 상위 노드(평가자)의 중요 정보 링크 지수(허브 지수)는 같은
   하위 노드에 **링크한 순서**에 따라 결정되고, 하위 노드의 중요도(권위 지수)는
   링크한 상위 노드들의 허브 지수 합으로 결정되며, 신규 하위 노드에 상위
   노드들이 링크할 때마다 **실시간으로 갱신**된다.
2. **(청구항 2)** 허브 지수는 내가 링크한 뒤 다른 평가자가 추가 링크할 때마다
   1씩 증가한다.
3. **(청구항 3)** 가중형: 이후 링크하는 평가자들의 허브 지수를 합산하는 방식.
4. **(청구항 8)** 여러 대상에 대한 허브 지수는 대상별 지수의 **평균**으로
   산출한다 → 무분별한 추천은 평균을 깎는 자기 조절 구조.

기술적 효과: 클라인버그(HITS)·페이지랭크와 달리 수렴을 위한 반복 연산이
불필요해, 실시간으로 쏟아지는 새 콘텐츠(뉴스·댓글)의 동적 평가가 가능하다.

## 등록특허 10-0952391 — 가치 분석 시스템

- 명칭: 인터넷 네트워크에서 콘텐츠의 평가에 따른 가치 분석 시스템, 방법 및 기록매체
- 출원: 10-2005-0031126 (2005-04-14) / 등록: 2010-04-05 / 공고: 2010-04-14
- 발명자: 홍정수 / 당시 특허권자: 에스케이커뮤니케이션즈(주)

핵심 청구: 평가 순서에 따라 평가자의 허브 지수를 산출하고, 평가자들의 허브
지수로 콘텐츠의 권위 지수를 산출하는 가치 분석 서버(청구항 1·9). 권위 지수에
따른 콘텐츠 노출 조정(청구항 6·14), 허브 지수 상위 사용자에게 편집 기능
부여(청구항 7·15). 여론 형성 구조(필터링·캐스팅·뉴스 네트워크)에의 적용.

## 이 시스템에서의 확장

특허의 "링크 순서"는 2005년 구현에서 서버 타임스탬프에 의존했다.
본 시스템의 줄서기 DAG(`src/weave/queue.js`)에서는 뒤에 서는 사람의 서명이
앞사람 항목의 해시를 덮으므로 **순서 자체가 암호학적으로 증명**된다.
즉 특허의 허브 지수가 위조 불가능한 분산 평판으로 승격된다
(`src/weave/insight.js`). 이 "서명 사슬로 증명되는 링크 순서 + 허브 지수"의
결합은 원 특허에 없는 새로운 기술적 구성이다.
