임기 없는 실시간 민주주의

투표로 왕을 뽑는다고 민주주의가 되지 않듯, 몇 년에 한 번의 선출로 민의가 대변되지 않습니다. 여기서는 지지가 언제든 철회되고, 모든 결정의 지위가 매 순간 다시 계산됩니다 — 서버 없이, 시민의 브라우저끼리 직접.

지금 참여하기 — 이 브라우저가 곧 나의 시민 단말 설치·가입·다운로드 없음. 키는 당신의 기기에서 생성되어 기기를 떠나지 않습니다.

왜 만들었나

임기제 선출은 선출 시점의 여론을 임기 내내 고정합니다. 이 레그(lag)가 포퓰리즘, 책임 회피, 민의와 권력의 괴리를 낳습니다. 이 시스템은 레그를 제거합니다:

지위는 어디에도 저장되지 않는다.
지지가 빠지면 그 순간 채택도 사라진다. 권력이 아니라 이의 제기 가능성이 상설화된 체제.

지지는 줄서기다

의견 뒤에 줄을 서며 앞사람을 서명으로 확인합니다. 참여가 곧 검증 — 줄이 길수록 기록을 지우기 어렵습니다. 반대도 똑같은 줄입니다.

안목이 평판이 된다

좋은 의견을 일찍 알아본 기록이 위조 불가능한 안목 지수가 됩니다. 헛발질하면 깎이고, 시간이 지나면 감쇠하며, 기본 1표는 불가침입니다.

서버가 없다

모든 데이터는 브라우저 사이를 WebRTC로 직접 흐릅니다. 이 사이트는 연결만 중개하는 전화번호부로, 누가 무엇을 지지하는지 볼 수 없습니다.

속도에 필터를 달았다

블라인드 초기 구간(정보 폭포 차단), 지속 다수(순간 격정 여과), 헌장 의제(다수의 폭정 방어), 추첨 배심(동원 불가능한 심사) — 민주주의 이론사의 처방을 코드로 구현했습니다.

어떻게 참여하나

  1. 참여하기를 누르고 이름(가명 가능)을 정합니다 — 키가 브라우저 안에서 생성됩니다.
  2. 전체 이슈에서 관심 있는 의제를 구독하거나, 새 이슈를 직접 공표합니다.
  3. 의견을 제안하고, 남의 의견에 자기 의견을 첨부해 지지·반대합니다. 생각이 바뀌면 언제든 줄에서 떠나면 됩니다 — 그 순간 집계가 바뀝니다.
  4. 바쁜 분야는 신뢰하는 시민에게 위임하세요. 직접 참여하는 순간 위임보다 우선하고, 언제든 회수됩니다.

같은 주소를 연 모든 브라우저(다른 기기 포함)가 서로 직접 연결된 하나의 시민 네트워크가 됩니다.

상시 노드를 운영하고 싶다면

브라우저 피어만으로도 참여는 충분하지만, 네트워크의 기억을 지키는 상시 노드(터미널 클라이언트)를 운영할 수 있습니다:

git clone https://github.com/wilcoco/decentraldemo
cd decentraldemo
node bin/peer.js --name 홍길동 --port 4001   # 상시 P2P 노드

소스 전체가 공개되어 있으며 외부 의존성이 없습니다(Node.js 18+). 집계 규칙은 브라우저와 노드가 같은 코드를 쓰므로, 누구든 모든 계산을 재검증할 수 있습니다.

정직한 안내

이것은 데모입니다.

신원은 현재 개방 등록이라 1인 다계정을 막지 못합니다(실제 운용에는 본인인증 → 익명 가명 발급의 2단 구조가 들어갑니다). 키는 브라우저 저장소에 보관되고, 서로 다른 통신사 NAT 뒤의 기기 간에는 연결 중계(TURN)가 추가로 필요할 수 있습니다. 설계 철학·보안 분석·민주주의 이론 검토 전문은 저장소의 docs/에 있습니다.
